This directory contains some generic Java list data structures like LinkedList, ArrayList and Stack.
All the list data structures were built from scratch. These structures can be used to store anytype of data.

References Used:
1) Data Structures and Algorithm Analysis in Java by Mark Allen Weiss. 
2) Ooracle Java APIs

Language Used: Java
Compiler/Running environment: Java Development Kit (JDK) is required on the machine in which you
	want to run the program. Java Runtime Environment (JRE) should be fine to run the program as
	well.

Download Java SDK for your architecture using the following URL: 
http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
